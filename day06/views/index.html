art-template学习测试页面

基础变量

标准写法：{{str}}

原始写法：<%= str %>

条件判断

标准写法： {{if(age > 18)}} 成年了 {{else if(age == 18)}} 刚好成年了 {{else}} 未成年 {{/if}}

原始写法： <%if(age > 18){%> 成年了 <%}else if(age == 18){%> 刚好成年了 <%}else{%> 未成年 <%}%>

循环操作

标准写法：

    {{each arr}}
  • {{$index}} - {{$value}}
    • {{/each}}
    {{each arr val key}}
  • {{key}} - {{val}}
    • {{/each}}

原始写法：

    <%for(var i = 0; i < arr.length;i++){%>
  • <%=i%> - <%=arr[i]%>
    • <%}%>

模版文件包含

标准写法：{{include "./inc.html"}}

原始写法：<%include("./inc.html")%>
